# Form Validation Project

This is a simple web-based form validation project built using HTML, CSS, and JavaScript.

## Features

- **Client-Side Validation**: Ensures all required fields are filled out before submission.
- **Password Complexity Check**: Enforces a strong password policy (at least 8 characters, with uppercase, lowercase, number, and special character).
- **Password Visibility Toggle**: Allows users to toggle password visibility to see what they are typing.
- **Immediate Feedback**: Displays error messages when fields are left empty or do not meet requirements.
- **Success Message**: Shows a confirmation message upon successful form submission.

## Technologies Used

- **HTML5**: For the structure of the form.
- **CSS3**: For styling the form and validation messages.
- **JavaScript (Vanilla)**: For handling the form validation logic.
- **FontAwesome**: For icons (e.g., password visibility toggle, success checkmark).

## How to Run

1. Clone the repository or download the source code.
2. Open `index.html` in any modern web browser.
3. Try filling out the form to see the validation in action!
